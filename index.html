    Portabilidade

    Backup

    Os dados deste site ficam salvos apenas neste navegador. Para usar em outro computador, celular ou tablet, exporte um backup JSON e salve no Google Drive. Depois, importe esse arquivo no outro dispositivo.

    Não compartilhe seu arquivo de backup com terceiros se ele contiver dados pessoais ou informações de estudo.

    Como rodar e publicar

    1. Baixe ou clone este repositório.
    2. Abra o arquivo index.html no navegador. Não é necessário instalar dependências.
    3. Os dados ficam no navegador usado; para testar do zero, use o botão Limpar dados.
    4. Para publicar no GitHub Pages, envie os arquivos para um repositório no GitHub.
