    Como rodar e publicar

    1. Baixe ou clone este repositório.
    2. Abra o arquivo index.html no navegador. Não é necessário instalar dependências.
    3. Para publicar no GitHub Pages, envie os arquivos para um repositório no GitHub.
    4. No GitHub, acesse Settings → Pages, selecione a branch principal e a pasta raiz, depois salve.
    5. Após alguns minutos, o GitHub exibirá a URL pública do site.
